# full-stack-notes-app

A full-stack web application built with Node.js and Express that allows users to manage their daily thoughts and tasks. This project demonstrates the integration of a frontend interface with a backend server to perform persistent data operations.

## Features

- **Create Notes:** Users can input new text and save it to the list.
- **View Notes:** All saved notes are rendered directly to the dashboard.
- **Delete Notes:** Users can remove notes from the database with a single click.
- **Persistent Storage:** All notes are saved to a `data.json` file on the server, ensuring data remains available after refreshing.
